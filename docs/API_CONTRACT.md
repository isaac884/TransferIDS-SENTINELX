# API Contract

All product APIs are versioned under `/api/v1`.

- `POST /api/v1/auth/login`
- `GET /api/v1/dashboard/summary`
- `POST /api/v1/detection/run`
- `POST /api/v1/intake/canonical-flows`
- `POST /api/v1/intake/upload`
- `GET /api/v1/events`
- `DELETE /api/v1/events/clear-dev-only`
- `GET /api/v1/incidents`
- `POST /api/v1/response/actions`
- `GET /api/v1/metrics/summary`
- `GET /api/v1/review/history`
- `POST /api/v1/review/decision`
- `GET /api/v1/integrations/health`
- `GET /api/v1/integrations/model-status`
- `GET /api/v1/integrations/database-status`
- `WS /api/v1/ws/live`
